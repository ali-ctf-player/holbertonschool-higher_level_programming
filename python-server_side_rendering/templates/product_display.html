How to use this page:

Use query parameters to filter products:

  • /products?source=json - Show all products from JSON
  • /products?source=csv - Show all products from CSV
  • /products?source=json&id=1 - Show product with ID 1 from JSON
  • /products?source=csv&id=2 - Show product with ID 2 from CSV

Try invalid sources like source=xml or non-existent IDs to see error messages.
